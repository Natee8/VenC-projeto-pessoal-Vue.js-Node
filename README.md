# VenC-projeto-pessoal-Vue.js-Node
VenCá é um marketplace de serviços pet on-demand que conecta tutores e cuidadores de forma rápida e simples. Usuários podem cadastrar pets, criar anúncios de passeios, alimentação, cuidados especiais ou urgentes, enquanto cuidadores visualizam oportunidades, enviam propostas e conversam via chat. Projeto focado em boas práticas e escalabilidade
